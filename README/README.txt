제작자 3101 고한결 
딥러닝 프로그램 사용법


AI.py 파일을 파이썬 IDE로 실행합니다. 


#layer 로 표기된 곳에 레이어에 대한 정의가 있습니다. 
거기에서 학습률 (Rate) 와 가중치 규제율 (Loss)를 설정할 수 있으나, 
현재 값이 최적화 된 값에 가깝습니다.

Layer 클래스 정의 밑에는 사용자 정의 함수들이 있습니다. 
실행의 편의를 위해 작성한 일종의 프리셋입니다.


#main 으로 표기된 곳에서 딥러닝 실행을 관리하게 됩니다.
L이라는 리스트 안에 레이어들을 생성할 수 있습니다.
레이어 생성은 
Layer(앞 레이어 크기, 자신의 크기, 레이어 타입) 
의 형식으로 생성할 수 있으며, 
레이어 타입에는 input, hidden, output 이 있고
아무것도 적지 않으면 hidden 이 됩니다.
인풋과 아웃풋 레이어는 꼭 필요하니 중간 레이어만 수정하는 것이 좋습니다.
(같은 폴더 내의 README 이미지 참조)
Train_data_limit으로 60000개의 훈련 데이터 중 
사용할 데이터의 범위를 설정할 수 있습니다.
Train_data_num은 한 번에 학습할 데이터 개수이고
Test_data_num은 테스트 시 사용할 데이터 개수입니다. 
Epoch는 총 학습 반복 횟수이고 1~10이 적당합니다.
Batch는 가중치 업데이트 간격입니다. 10이 적당합니다. 
test_during_train은 학습 중에 에러 테스트를 할 지 여부입니다.

실행하면 test 함수에 의해 처음과 끝의 에러를 알 수 있고, 
run 함수가 자동으로 학습을 실행합니다.

학습이 모두 끝났을 때 test_during_train = True 라면
엑셀 파일로 학습 경과가 기록됩니다.

폴더 내에 포함된 file.bmp파일은 사용자 테스트 파일입니다.
학습이 모두 끝났을 때 자동으로 사용자 테스트가 시작되는데,
이 파일에 그림을 그려 직접 입력할 수 있습니다
